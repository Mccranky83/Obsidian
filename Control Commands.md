| :------------------------ | :----------------: |

| next-suggestion           |      `<C-n>`       |
| previous-suggestion       |      `<C-p>`       |
| cursor-character-backward | `<C-b>` or `<C-o>` |
| cursor-character-forward  |      `<C-f>`       |

| cursor-line-end           | `<C-v>` or `<C-e>` |

| paste-history             |      `<C-y>`       |
| backward-switch-character |      `<C-t>`       |
| search-as-url             |      `<C-CR>`      |
